Ejercicio 27

Consigna:

Como resultado de una encuesta se recogen los siguientes datos: Sexo (H: hombre, M: Mujer), edad (en años) y altura (en centimetros).

Realice un programa para calcular e informar:

  1. El procentaje de mujeres mayores a 25 años.
  2. El promedio de edad de las mujeres.
  3. El promedio de altura de las mujeres.
  4. El promedio de altura de los hombres.
  5. La menor edad ingresada.
  6. La mayor altura ingresada.
